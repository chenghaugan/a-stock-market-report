## 新闻数据复用（新增）

**⚠️ 重要：周报生成时优先复用日报新闻数据，仅补充未覆盖内容**

### 复用机制

```
1. 读取本周周一至周五的 daily news 文件（output/news/daily/news_YYYYMMDD.json）
2. 聚合所有新闻，按日期排序，去重（基于 URL）
3. 检查覆盖率：是否有重大事件遗漏
4. 如有遗漏，补充 2-3 次 Tavily 搜索（仅搜索未覆盖的内容）
5. 保存聚合后的新闻到 output/news/weekly/news_YYYY-WXX.json
```

### 复用优先级

| 来源 | 优先级 | 说明 |
|-----|-------|------|
| daily news（本周） | 最高 | 直接复用，节省 credits |
| 补充搜索 | 中等 | 仅覆盖遗漏的重大事件 |
| 完全重新搜索 | 最低 | 仅在无 daily news 时使用 |

### 补充搜索触发条件

当以下情况发生时，执行补充搜索：
- 本周某些日期没有 daily news 文件（非交易日）
- 覆盖检查发现重大宏观事件未覆盖（如政策发布、重要会议）
- 用户明确要求补充特定主题的新闻

### 命令示例

```python
# 使用 scripts/news_storage.py
from news_storage import aggregate_weekly_news, save_weekly_news

# 1. 聚合周度新闻（自动复用 daily news）
weekly_news = aggregate_weekly_news("2026-04-22")

# 2. 如需补充搜索（未覆盖的重大事件）
supplement = tavily_search("本周重大政策发布 2026-04")
weekly_news = aggregate_weekly_news("2026-04-22", supplement)

# 3. 保存聚合后的新闻
save_weekly_news(weekly_news)
```
